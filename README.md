# mwcom-playwright-reports

This repository contains Playwright test automation reports for the mwcom project.

## Overview

This project uses Playwright for end-to-end testing and generates comprehensive test reports.

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

```bash
npm install
```

### Running Tests

```bash
npx playwright test
```

### Viewing Reports

After running tests, view the generated HTML report:

```bash
npx playwright show-report
```

## Project Structure

```
├── tests/              # Test files
├── reports/            # Generated test reports
├── playwright.config.js # Playwright configuration
└── README.md          # This file
```
